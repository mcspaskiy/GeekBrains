Название приложения - Children Kanban.

В основу заложена методология "Agile". Agile, реорганизовывая и переосмысливая действия, связанные с повседневной жизнью детей, позволяет достигать поставленных целей гибким и направленным на развитие ребенка способами. Приложение предназначено  для постановки задач на день родителями детям по методике "Kanban".
Через приложение родитель имеет возможность создавать задачи для ребенка и назначать их на некоторый период времени. Далее родитель имеет возможность через это же приложение отслеживать движение этих задач по доске канбан. Каждой задаче родитель может назначать поинты, которые отображают важность задачи. Поинты суммируются за день и ведется статистический подсчет за все время. К устройству, на котором установлено приложение для родителя, может быть подключено устройство ребенка. Родительское приложение генерирует QR-код, данный код сканирует приложение ребенка и регистрируется в системе как дочернее устройство.
Ребенок на своем устройстве видит задачи и может менять их состояние путем перетаскивания по доске. В качестве связывающего слоя между приложением для родителя и приложением для ребенка выступает Firebase Realtime Database. В базе устанавливается связь по идентификатору устройства родителя и ребенка и происходит синхронизация статуса задач.
Таким образом, родитель генерирует задачи и отслеживает их статус, а ребенок выполняет и меняет статус.

Сущности:
Parent(Родитель)
Child(Ребенок)
Task(Задача)
Board(Доска)

Ограничения и связи:
Родитель может создавать сколько угодно задач.
Родитель связан с сущностью задача через создание
У ребенка может быть сколько угодно задач.
Доска содержит в себе задачи
Родитель и ребенок связаны через доску

Функционал:
Родитель может подключить дочернее устройство.
Родитель может создавать/изменять/удалять задачу.
Ребенок может двигать задачу по доске.
